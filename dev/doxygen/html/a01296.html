Detailed Description

template<typename BH, typename FER = FERequirements<>, bool useEigenRef = false, bool useFlat = true>
class Ikarus::KirchhoffLoveShell< BH, FER, useEigenRef, useFlat >

This class represents Kirchhoff-Love shell finite elements.

Template Parameters
BHThe basis handler type for the finite element.
FERThe type representing the requirements for finite element calculations.
useEigenRefA boolean indicating whether to use Eigen references for efficiency.
useFlatA boolean indicating if the type of the underlying basis is of the flat or the untouched version.

Detailed Description

template<typename BH, bool useFlat = true, typename FER = FERequirements<>, bool useEigenRef = false>
class Ikarus::FEBase< BH, useFlat, FER, useEigenRef >

A single class which can be used by the finite elements to get different information from the local view of the element. It works for both the flat and untouched version of the basis.

Template Parameters
BHThe type of the basis handler.
useFlatA boolean indicating if the type of the underlying basis is of the flat or the untouched version.
FERThe requirements for the finite element.
useEigenRefA boolean flag indicating whether to use Eigen references.

template<typename B, typename FER = FERequirements<>, bool useEigenRef = false>

This class represents Kirchhoff-Love shell finite elements.

FERThe type representing the requirements for finite element calculations.
useEigenRefA boolean indicating whether to use Eigen references for efficiency.

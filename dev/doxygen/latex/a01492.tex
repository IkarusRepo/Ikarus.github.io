class Ikarus\+::\+Dirichlet\+Values$<$ Basis\+\_\+, Flags\+Type\+\_\+ $>$}
The \mbox{\hyperlink{a01492}{Dirichlet\+Values}} class provides functionalities for fixing degrees of freedom and storing inhomogeneous Dirichlet boundary conditions. It supports fixing degrees of freedom using various callback functions and stores functions for inhomogeneous Dirichlet boundary conditions.

This function registers \mbox{\hyperlink{a00375}{Python}} bindings for a \mbox{\hyperlink{a01720}{Dirichlet\+Values}} class, allowing it to be used in \mbox{\hyperlink{a00375}{Python}} scripts. The registered class will have an initializer that takes a {\ttfamily Basis} object. It exposes several member functions to \mbox{\hyperlink{a00375}{Python}}\+:
\begin{DoxyItemize}
\item {\ttfamily fix\+Boundary\+DOFs(f)}\+: Fixes boundary degrees of freedom using a user-\/defined function {\ttfamily f} than can be called with the following arguments\+:
\begin{DoxyItemize}
\item with the boolean vector and the global index.
\item with the boolean vector, the local index and the {\ttfamily Local\+View}.
\item with the boolean vector, the local index, the {\ttfamily Local\+View} and the {\ttfamily Intersection}.
\end{DoxyItemize}
\item {\ttfamily fix\+DOFs(f)}\+: Fixes boundary degrees of freedom using a user-\/defined function {\ttfamily f} with the basis and the boolean vector as arguments.
\item {\ttfamily set\+Single\+DOF(i, flag\+: bool)\+: Fixes or unfixes DOF with index i -\/}is\+Constrained(i){\ttfamily \+: Checks whether index i is constrained -\/}reset()\`{}\+: Resets the whole container
\end{DoxyItemize}

The following properties can be accessed\+:
\begin{DoxyItemize}
\item {\ttfamily container}\+: the underlying container of dirichlet flags (as a const reference)
\item {\ttfamily size}\+: the size of the underlying basis
\item {\ttfamily fixed\+DOFsize}\+: the amount of DOFs currently fixed
\end{DoxyItemize}

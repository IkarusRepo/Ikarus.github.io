\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename BH, bool use\+Flat = true, typename FER = FERequirements$<$$>$, bool use\+Eigen\+Ref = false$>$\newline
class Ikarus\+::\+FEBase$<$ BH, use\+Flat, FER, use\+Eigen\+Ref $>$}
A single class which can be used by the finite elements to get different information from the local view of the element. It works for both the flat and untouched version of the basis.


\begin{DoxyTemplParams}{Template Parameters}
{\em BH} & The type of the basis handler. \\
\hline
{\em use\+Flat} & A boolean indicating if the type of the underlying basis is of the flat or the untouched version. \\
\hline
{\em FER} & The requirements for the finite element. \\
\hline
{\em use\+Eigen\+Ref} & A boolean flag indicating whether to use \mbox{\hyperlink{a00334}{Eigen}} references. \\
\hline
\end{DoxyTemplParams}
